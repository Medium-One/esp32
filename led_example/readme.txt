  The C firmware is created for the following ESP32 boards. 
     1. Adafruit Huzzah ESP32  (the onboard Red LED near the USB header)
     2. SparkFun ESP32 Thing (the onboard Blue LED at the middle of the board)
     3. Expressif ESP32 DevKitC v4 (External LED and resistor required. Connect them to GPIO<16> )
